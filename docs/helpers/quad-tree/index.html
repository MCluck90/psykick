QuadTree

Keeps track of physical objects and finds collisions
new QuadTree( options ) [Deprecated: Use DataStructures.QuadTree]

Parameters

options Object
x Number X position of the world
y Number Y position of the world
w Number Width of the world
h Number Height of the world
cellSize Number Size of a cell 100

Properties

x Number X position of the world
y Number Y position of the world
w Number Width of the world
h Number Height of the world
cellSize Number Size of a cell
children QuadTree[] Each node has 4 children until reaching the minimum size
entities Entity[] Collection of entities in the cell

Methods

addEntity( entity , body )

Adds an entity to the tree

entity Entity Entity to add to the tree
body RectPhysicsBody Rectangular physics body. Used during recursion entity.getComponent('RectPhysicsBody')

removeEntity( entity , body )

Removes an entity from the tree

entity Entity Entity to remove from the tree
body RectPhysicsBody Rectangular physics body. Used during recursion entity.getComponent('RectPhysicsBody')

moveEntity( entity , deltaPosition )

Moves an Entity and updates it's position in the tree

entity Entity Entity that will be moved
deltaPosition Object Position changes
x Number X position change
y Number Y position change

Entity[] getCollisions( entity , body )

Returns all entities the given entity is colliding with

entity Entity Entity to check collisions against
body RectPhysicsBody Physics body from Entity. Used during recursion entity.getComponent('RectPhysicsBody')
