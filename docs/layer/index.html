Layer

A layer houses a set of systems which are updated/drawn on each frame
new Layer( options )

Parameters

options Object
id Number Unique ID assigned by the world
container HTMLElement Element which houses the layer
serverMode Boolean If true, a canvas will not be made false

Properties

id Number Unique ID assigned by the world
container HTMLElement Element which houses the layer
renderSystems RenderSystem[] Collection of rendering systems
behaviorSystems BehaviorSystem[] Collection of behavior systems
active Boolean If true, the layer is updated true
canvas HTMLElement|null The layers personal canvas
stage PIXI.Stage pixi.js stage for rendering
scene PIXI.DisplayObjectContainer A container for all of the renderable objects. Allows the camera to work
renderer PIXI.WebGLRenderer|PIXI.CanvasRenderer Used for rendering the scene
_serverMode Boolean If true, avoid all draw operations
visible Boolean If true, the layer is visible true

Methods

removeCanvas( )

Removes the canvas to ensure no additional drawing is done

restoreCanvas( )

Puts the layers' canvas back in the container if it was removed

setZIndex( zIndex )

Sets the depth layer index. Used by the @World

zIndex Number Desired z-index

addSystem( system )

Adds a new system to the layer

system System New system

removeSystem( system )

Removes a system from the layer

system System System to remove

draw( delta )

Render the layer

delta Number Time since previous update

update( delta )

Update the layer

delta Number Time since previous update
