World

The game world
static World

Methods

init( options )

Initializes the world

options Object
canvasContainer HTMLElement|String Main container for the game. Defaults to using <div id='psykick'></div> psykick
width Number Width of the game 800
height Number Height of the game 600
backgroundColor String Background color #000
serverMode Boolean If true, no drawing will occur false

Entity createEntity( )

Creates a new Entity

Layer createLayer( )

Creates a new Layer

Boolean pushLayer( layer )

Pushes a Layer on to the top of the draw stack
Returns true if the Layer was not on the stack

layer Layer Layer to place on the top of the stack

Layer|null popLayer( )

Pops and returns the Layer on the top of the draw stack

update( delta )

Updates the world

delta Number Time since previous update

draw( delta )

Draws the world

delta Number Time since previous update

addEventListener( eventType , listener )

Adds a new event listener
Possible events: beforeUpdate, afterUpdate, beforeDraw, afterDraw

eventType String Event to listen for
listener Function Callback

removeEventListener( eventType , listener )

Removes an event listener

eventType String Event the listener is listening to
listener Function Callback

removeAllListeners( eventType )

Removes all listeners for a given event

eventType String Event to remove all callbacks from

reset( )

Resets the state of the world
All Layers are removed
ID counters return to 0
